# 🎂 Happy Birthday Chuto Apu

A small birthday surprise website created to celebrate my sister's special day.  
This project shows birthday wishes and some beautiful memories with interactive animations.

## 🌐 Live Website
👉 https://happybirthdaychutoapu.netlify.app/wish

## 📌 Features
- 🎉 Beautiful birthday wish page
- 📸 Memory gallery with photos
- 🔄 Flip cards to reveal memories
- 🎵 Smooth animations and effects
- 📱 Responsive design for mobile and desktop

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📷 About the Project
This website was created as a personal project to make my sister feel special on her birthday.  
The site includes a greeting page and a collection of our old memories presented in an interactive way.
